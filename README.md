## Installation need:

### Linux:

* Python:

    1-Update Package List:

         sudo apt update

    2-Install Python 3:

        sudo apt install python3

    3- Verify installation:

        python3 --version

* Pycodestyle.py

    1-Update Package list:

       sudo apt update

    2-install pycodestyle:

       sudo apt-get -y install pycodestyle
